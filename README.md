# Listing Schedule Selector

A React component converted from Bubble.io for selecting days/nights for rental property listings.

## Quick Start

\`\`\`bash
npm install
npm run dev
\`\`\`

See SETUP_GUIDE.md for detailed instructions.

## Features

- Day selection with contiguity validation
- Multiple rental types (Monthly, Weekly, Nightly)
- Dynamic pricing calculations
- Error handling and validation
- Supabase integration
- All custom states from Bubble implementation
